## 作業說明及相關資源

在這份作業中，你需要應用上一堂課所學到的知識，使用瀏覽器的網頁開發者工具來找到範例網頁中指定元素的顏色。請下載整份資料夾後預覽 starter 資料夾的 [index.html](./index.html) 根據下述 [題目](./%E9%A1%8C%E7%9B%AE.png) 敘述透過 [作業提交範本](#作業提交範本) 提交你的答案，請注意答案**僅能**為 hex 色碼。
